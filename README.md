# -Guerreiros-da-Luz
Aplicativo bíblico gamificado com foco em ensino, discipulado e diversão, desenvolvido para  igrejas, EBD, crianças, jovens e adultos.
## Visão do Projeto: Guerreiros da Luz é uma plataforma digital que transforma o ensino bíblico em uma experiênciainterativa, com missões, perguntas, temporadas e recompensas.
Público Alvo: Crianças (modo Kids)-Jovens - Adultos - Profesores e líderes cristãos.
## Status do Projeto: Em desenvolvimento - Estrutura inicial criada - Conteúdoem expansão por temporadas.
